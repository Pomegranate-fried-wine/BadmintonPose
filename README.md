# BadmintonPose
A CV-based biomechanical analysis system for badminton using SAM and 3D reconstruction.

---

# 项目简介：基于单目视觉的羽毛球赛事三维场景重建与球员动作分析系统

## 🌟 项目定位

本项目旨在打破职业级体育分析对高昂硬件（如多目高速摄像机、穿戴式传感器）的依赖。通过**固定单目相机**采集视频，结合计算机视觉（CV）**与**具身智能（Embodied AI）相关技术，实现从底层物理像素到高层竞技语义的深度转化。

## 🛠 技术路线

项目融合了前沿的 3D 重建算法与鲁棒的多目标追踪框架：

1. **3D 场景重建与几何标定**：
* 利用 **VGGT (Visual Geometry Grounded Transformer)** 与物理几何约束，将 2D 视频流映射至 3D 空间坐标系。
* 通过霍夫变换与张正友标定法，实现对标准羽毛球场地的厘米级精度重建。


2. **多模态目标感知**：
* **检测与分割**：采用 **YOLO** 配合 **SAM3 (Segment Anything Model 3)**，在复杂动态背景下实现对羽毛球及球员的像素级分割。
* **姿态估计**：利用 **Sports2D** 算法提取人体 17 个核心关节的132个关键点，建立生物力学模型。


3. **时序分析与物理轨迹补全**：
* 针对羽毛球体积小、速度快的特点，引入卡尔曼滤波（Kalman Filter）进行多帧关联。
* 结合空气动力学物理模型，对遮挡或丢失帧进行轨迹逻辑补全。


4. **认知决策闭环（LLM Integration）**：
* 将提取的运动矢量（初速度、击球角度、步法频率）进行结构化处理。
* 利用 **Qwen (通义千问)** 大模型作为决策大脑，输出专业级技术诊断报告。



## 📈 核心功能

* **全自动 3D 轨迹还原**：可视化羽毛球在三维空间中的落点、弧度及飞行速度。
* **生物力学指标量化**：精准解析球员击球时的关节角度、角速度及发力因果链。
* **AI 专家复盘**：自动生成结构化复盘建议，包括战术失误归因与针对性训练指导。
